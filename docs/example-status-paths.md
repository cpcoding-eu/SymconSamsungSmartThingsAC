# Typische SmartThings Status-Pfade

Diese Pfade dienen nur als Orientierung. Der genaue Status hängt vom Samsung-Modell und vom SmartThings Edge Driver ab.

```text
switch.switch.value = on/off
thermostatCoolingSetpoint.coolingSetpoint.value = 22
temperatureMeasurement.temperature.value = 23.4
airConditionerMode.airConditionerMode.value = cool/heat/dry/fanOnly/auto
airConditionerFanMode.fanMode.value = auto/low/medium/high/turbo
relativeHumidityMeasurement.humidity.value = 45
custom.airConditionerOptionalMode.acOptionalMode.value = off/quiet/windFree/sleep
```
